### 2025/11/29

首先讲解了“C 大调音阶图”：
```
1弦：3.|---4.----|-------|---5.----|
2弦：7 |---1.----|-------|----2.---|
3弦：5 |-------|----6---|-------|
4弦：2 |-------|----3---|----4---|
5弦：.6|-------|----.7---|---1----|
6弦：.3|----.4---|-------|----.5---|
```
> 注：受限于格式，上述的 `.3` 表示`低音 3`，`3.` 表示`高音 3`~

这里再说明下音阶和音符的关系，一个 `1`，我们管它叫做音符。如果把音符按照某种排列组合起来，就称为音阶，比如：`1234567` 就是一种音阶，通常
叫做“音阶上行”；`7654321`

接着老师讲解了“十二平均律”，音乐的 7 个音符在吉他同一根琴弦上的分布：
```
|----1---|-------|----2---|-------|----3---|----4---|-------|----5---|-------|----6---|-------|----7---|----1.---|
```

这里我们说，同一根琴弦上的音符：
- `1` 和 `2`，`2` 和 `3`，`4` 和 `5`，`5` 和 `6`，`6` 和 `7`，这种中间隔一个品格的关系，叫做**“全音”**；
- `3` 和 `4`，`7` 和 `1`，这种相邻品格的关系，叫做**“半音”**

这里我们也可以看到一个规律：**同一根琴弦上，品位越高，音就越高**

那我们有了“十二平均律”，我们就可以根据“C 大调音阶图”，推出每根琴弦上的音符了～

那你说 `1` 和 `2` 是全音关系，中间隔的品位是什么音呢？它可以是`升 1`，也可以是`降 2`。音乐中“升”用符号 `#` 表示，
“降”用符号 `b` 表示，所以我们可以把这个品位的音记作 `#1/b2`。

有了这个知识，我们重新再把琴弦上的图画一遍：
```
|----1---|---#1/b2----|----2---|----#2/b3---|----3---|----4---|---#4/b5----|----5---|----#5/b6---|----6---|---#6/b7----|----7---|----1.---|
```

这样的话，琴弦上的每个品格都有了自己的音符啦～
